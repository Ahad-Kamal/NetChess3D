# Net Chess 3D

This project replicates the classic game of Chess, with (To-be-added) local network support. 
The Project can be played by pulling down the Run folder and launching the executable file. You can view the code for this project in the Code folder. 
This project was made in my own custom game engine, which you can find the repository for here: https://github.com/Ahad-Kamal/Engine

Chess pieces can be moved by pressing ~ and using the ChessMove command. Ex. ChessMove from=e2 to=e4.

During the course of this project, I learned a lot about hlsl shader coding, which I used to achieve lighting. To that end, there are various shader debug modes you can view by pressing the 0-9 keys.
As for why there's a giant Luigi in the backgorund... hes's just there for decoration.
